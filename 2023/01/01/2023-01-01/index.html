2023-01-01

莫名妙的又過了一年，新年真的越來越懶了，剛剛才發現去年竟然連照片都沒拍 XDD，今年一定拍！

去年給T姊算命後他說我 2022 年的工作會超忙，忙到沒時間談戀愛的那種，果真去年七月開始有金融業13家銀行的聯合攻防演練，那真的要死人
組內基本上技術部分只有我在扛，也一如去年初說的一樣，我在 12/07 提了離職，預計 1/7 離開，太累了，那也一如去年說的一樣，我不知道下一家在哪XDD，裸辭！
不過有另外得到的回饋是今年會有點桃花…..? 聽聽就好聽聽就好，不抱期待

回想一下去年幹了什麼事情，要說前年受到獅子很多影響的話，去年受到 RD 很多的照顧，受到他們的影響比較多，沒辦法 SST 真的獲得有限了
今年主要買了兩隻 yubikey，兩隻配合 Cloudflare 的 coupon 含運收 800，對比原價一隻破千真的很划算
之後就開始研究個人的資安，把重要的服務，像是 1password 和 google 和 facebook 等帳號都鎖上 security key 了，有關 security key 的應用我單獨發一篇出來了
再來就是跟同事小T買了他二手的 roouter ( hEX - MikroTik )，目前套了一些比較基本的 firewall rule，像是 block ip scanner 之類的，還有就是把 router dns 設定成 nextdns，方便做管控，目前卡在我想要弄 wireguard 讓身在咖啡廳的我只需要 ipad 就能連回家裡的 windows 進行操作，或是連回家裡的樹莓派弄手遊，目前知道的應該就是走 ZeroTier 或是 Tailscale，今年不知道是否能搞起來

說到今年真的拿到不少東西，除了 yubikey 以外還有被推坑買了 ipad pro m1 大概花了 19500…
還有拿到一個免費的 rpi400，can 爸給我的
還有花 500 拿到的那台 router
買了 iphone 13 mini (2020 買的 iphone se 在去年過年的時候摔壞了…)
訂閱了 ocf 每個月 300
買了一個短域名，並且把他的 dns 托給 Cloudflare，今年應該也會玩不少 CF 的功能
加入了 uccu 並且參與了線下聚會
還有12月拿到熱騰騰的 flipper zero，海豚很可愛，他也是我今年想玩的玩具—實體安全
還有為了上面那隻海豚特別去光華買了 nrf24 模組（想玩 mousejacking）

去年的文章產出量真的很少，比較多是補充既有的文章內容，畢竟舊坑都補不完了還要忙著開新坑….
再來就是工作太累了，不得不說在 LKC 的那個案子時期比勤業還累…，很多時候都忙到半夜

今年感覺又更認識自己了，可以看到上面的一堆東西都跟 pwn 無關
對自己的認知轉變就是，感覺對資訊（資安）的熱情還在，但並不僅止於 pwn，對於 pwn 去年還是生了一篇 brop，然後 file structure exploit 還卡在 libc 2.27 怎麼突破＠＠
今年很怕這樣學一學最後變成四不像，什麼都不專精導致沒人要;w;

再看看吧今年，希望今年能達成包兩萬紅包給爸媽的願望（感覺就是以前我完全不會說的話
